# EMS Default Manuscript Theme

**Editorial Management System - Default Manuscript Theme**

A professional, clean theme for scientific journals built on the OJS Default Theme framework.

## Features

- Clean, manuscript-style layout with boxed design
- Custom hero section for journal homepage
- EMS brand favicon and logo integration
- Customizable accent color
- Responsive design for mobile, tablet, and desktop
- Custom navigation without search functionality
- Professional footer with About and Privacy links
- Dynamic copyright notice

## Installation

1. Copy this theme folder to your OJS installation's `plugins/themes/` directory
2. Activate the theme in the admin dashboard under Settings → Website → Appearance
3. Configure theme options including accent color and journal branding

## Customization

The theme supports:
- Accent color customization
- Base color customization from parent theme
- Journal thumbnail display on homepage
- Custom footer branding

## Requirements

- OJS 3.3 or higher
- Parent theme: Default Theme Plugin
